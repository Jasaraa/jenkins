🚀 CI/CD Pipeline Project

Automated Deployment using Jenkins, GitHub & AWS

📌 Project Overview

This project demonstrates an end-to-end Continuous Integration and Continuous Deployment (CI/CD) pipeline that automates application deployment from source code to a live server.

🛠️ Technologies Used

  • GitHub – Version Control System
  • Jenkins – Automation Server
  • AWS EC2 – Hosting Environment
  • Apache – Web Server

⚙️ CI/CD Workflow

Code Commit → GitHub → Jenkins Pipeline → AWS EC2 → Live Deployment

🚀 Key Features

  • Automated deployment on code push
  • Eliminates manual intervention
  • Ensures consistency and reliability
  • Fast and scalable deployment process
